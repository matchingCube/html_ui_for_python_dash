PROCUREMENT CONTRACTS IN COLOMBIA

05/12/2019
30/12/2019
MOTIVATION

Colombia compra eficiente is the ruling body for public procurement contracts in Colombia. Our aim in this project is to build a classification model that would allow us to detect atypical data in public contracts. We gathered our data from the SECOP I and II public contract databases.

IMPACT

It is our hope that this project will help make public resource allocation more efficient, and that it would increase oversight of how Colombians tax money is being used.
